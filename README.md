# eigengram-python

> Persistent KV cache semantic fingerprinting for large language models.

**Status:** Pre-release. Protocol under active development.

The Eigengram Protocol enables cross-session, cross-model retrieval
of LLM computation states using Fourier-decomposed KV cache fingerprints.

- [Specification](https://github.com/eigengram-org/eigengram-spec)
- [Python SDK](https://github.com/eigengram-org/eigengram-python)
- [HuggingFace](https://huggingface.co/eigengram)
